# mber_base
Base for games from M-BER (_ember_, reference to MBIR) studio

# Dependencies
## Main
- Godot 4.6

## Android
- JDK-17
- Android studio (with the following SDK Tools)
  + Android SDK Build-Tools
  + Android SDK Platform-Tools
  + Android SDK Command-line Tools
  + CMake
  + NDK
